# Acadify Frontend

Plantilla mínima de frontend para Acadify usando Vite + React + TypeScript y TailwindCSS.

Comandos:

- Instalar dependencias:
  npm install
- Ejecutar en modo desarrollo:
  npm run dev

Características incluidas:
- Tailwind configured (dark mode via class)
- Theme toggle (persisted in localStorage)
- Framer Motion for entrance animations
- Paleta: morado principal (#6B21A8), verde acento (#16A34A), blanco/negro

Notas:
- Para cambiar los colores, edita `tailwind.config.cjs` y las variables en `src/index.css`.
